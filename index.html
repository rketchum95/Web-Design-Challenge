Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes the further you move from the equator. This analysis was done by pulling weather related date from 500+ cities using the OpenWeatherMap API.

Matplotlib was used to plot various weather factors, including temperature, cloudiness, humidity, and wind speed, against the latitude. These charts were helpful in during the analysis portion of the project.
